# Credit Card Fraud Analysis

## Project Overview

This project analyzes credit card transactions to identify fraud patterns using Python.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Objectives

- Explore transaction data
- Analyze fraud distribution
- Visualize fraud patterns
- Generate business insights

## Dataset

Credit Card Fraud Detection Dataset from Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## Key Insights

- Fraud transactions are extremely rare.
- Most transactions are legitimate.
- Data visualization helps identify fraud patterns.
- Correlation analysis highlights important features related to fraud.
